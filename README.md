This repository is configured to automatically deploy to GitHub Pages! 

### How It Works

1. **Automatic Deployment**: Every time you push to the `main` or `master` branch, the site will automatically deploy to GitHub Pages.
2. **Manual Deployment**: You can also trigger deployment manually from the Actions tab in GitHub.

### Setup Instructions

To enable GitHub Pages for your repository:

1. Go to your repository settings
2. Navigate to **Pages** section
3. Under **Source**, select **GitHub Actions** as the build and deployment method
4. That's it! Your site will be available at `https://journeyto0.github.io/XpenseWise/`

### Adding Your Files

Simply add your HTML, CSS, JavaScript, and other static files to the repository:

- Replace or modify `index.html` with your expense tracking application
- Add CSS files for styling
- Add JavaScript files for functionality
- Include images and other assets

All files in the repository root will be deployed to GitHub Pages automatically when you push to the main branch.

### Local Development

To test your site locally before pushing:

1. Clone the repository
2. Open `index.html` in your browser
3. Make your changes
4. Commit and push to deploy
